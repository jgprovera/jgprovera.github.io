Gestor de tareas en Python

Aplicación de terminal desarrollada en Python para gestionar tareas mediante almacenamiento persistente en archivos de texto. El programa permite registrar nuevas tareas, visualizar tareas existentes y eliminarlas de forma interactiva utilizando un menú en línea de comandos.

El proyecto incorpora funciones modulares, manejo de archivos, validación básica de entradas, estructuras de datos basadas en listas y persistencia de información mediante operaciones de lectura y escritura sobre el sistema de archivos.

Avances: implementación de menú interactivo, almacenamiento persistente en archivo de texto, numeración automática de tareas mediante enumerate(), eliminación de tareas, manejo de excepciones básicas y publicación del proyecto como repositorio público documentado.

Tecnologías: Python · Linux (Arch Linux) · Git · GitHub

Repositorio: github.com/jgprovera/gestor-tareas

Organizador de archivos en Python

Herramienta de automatización desarrollada en Python para la clasificación y organización de archivos según su extensión. El programa inspecciona el contenido de un directorio, identifica el tipo de cada archivo y ejecuta su reubicación automática dentro de una estructura de carpetas generada dinámicamente.

La implementación incorpora funciones modulares, estructuras de datos basadas en diccionarios, condicionales, bucles, validación de rutas y operaciones sobre el sistema de archivos mediante los módulos estándar os y shutil.

Avances: desarrollo de una estructura modular con funciones específicas, clasificación automática por extensión, creación dinámica de carpetas, validación básica de rutas y publicación del proyecto como repositorio público documentado.

Tecnologías: Python · Linux (Arch Linux) · Git · GitHub

Repositorio: github.com/jgprovera/organizador-archivos

Portfolio personal — provera.uy

Sitio web técnico desarrollado para centralizar proyectos, documentación y evidencia pública de trabajo. El portfolio funciona como punto de acceso a los repositorios y proyectos desarrollados, manteniendo una presentación profesional y verificable de las habilidades técnicas adquiridas.

El proyecto incluye desarrollo frontend con HTML y CSS, control de versiones mediante Git, publicación con GitHub Pages, configuración de dominio personalizado y despliegue continuo mediante repositorios públicos.

Avances: diseño inicial del sitio, publicación en GitHub Pages, integración con dominio propio provera.uy, configuración DNS, soporte para www y mejoras continuas en estructura y presentación.

Tecnologías: HTML · CSS · Git · GitHub · GitHub Pages · DNS
